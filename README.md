> A slack bot to provision github's repositories

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# run the code as in production environment
$ npm run start:prod
```

## Running the tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov

# all tests
$ npm run test:all
```
